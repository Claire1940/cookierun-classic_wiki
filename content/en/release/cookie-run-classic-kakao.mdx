<Section icon={History} title="The Legacy of Cookie Run Classic Kakao">

**Cookie Run Classic Kakao** (Korean: 쿠키런: 쿠키들의 오븐탈출 대작전!) represents the genesis of the Cookie Run universe. Originally launched on April 2, 2013, this version serves as the foundation for all subsequent titles like OvenBreak and Kingdom. Unlike the international LINE version which was discontinued in 2018, the Kakao version remains a living ecosystem with consistent updates and a dedicated player base in South Korea.

The game operates on the KakaoTalk social platform, which dictates much of its social functionality. While the core loop of jumping and sliding remains consistent, the Kakao version often serves as a testing ground for new mechanics and seasonal content before they are considered for other titles in the franchise. As of 2026, it stands as the longest-running active mobile runner in the region.
