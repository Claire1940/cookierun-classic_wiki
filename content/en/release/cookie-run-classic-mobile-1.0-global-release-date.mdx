<Section icon={Target} title="Core Gameplay and Beginner Strategy">

The gameplay following the **Cookie Run Classic mobile 1.0 global release date** remains faithful to the original formula. Your primary goal is to run as far as possible while collecting Jellies and avoiding the Witch's obstacles. The "Classic" version emphasizes precision over complex ability rotations, making it easy to pick up but difficult to master at high League levels.

| Mechanic | Description | Strategy Tip |
| :--- | :--- | :--- |
| **Jump** | Clears floor obstacles and gaps | Double-tap for a higher leap over stacked crates. |
| **Slide** | Passes under hanging obstacles | Hold to stay low; release quickly to resume running. |
| **Jellies** | The primary source of points | Follow the Jelly paths for the safest route through stages. |
| **Health (Energy)** | Depletes over time or upon impact | Prioritize **Health Potions** over rare Jellies to extend runs. |
| **Combi Bonus** | Synergy between Cookie and Pet | Match specific Pets to their partner Cookie for extra points. |
