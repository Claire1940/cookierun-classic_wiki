ใน คุกกี้รัน คลาสสิก การวิ่ง การกระโดด และการสไลด์เป็นเพียงครึ่งเดียวของการต่อสู้ เพื่อแย่งชิงพื้นที่ในตารางคะแนนลีกแบบเรียลไทม์ คุณต้องเชี่ยวชาญระบบสมบัติ สมบัติคือไอเทมแบบพาสซีฟและแอคทีฟที่สามารถอัปเกรดได้ ซึ่งจะมอบตัวคูณคะแนน การป้องกันด้วยโล่ สนามแม่เหล็ก หรือการสร้างเหรียญระหว่างการวิ่ง ไม่เหมือนกับคุกกี้และสัตว์เลี้ยงที่กำหนดพลังชีวิตพื้นฐานและทักษะหลัก สมบัติจะมอบตัวปรับแต่งที่ยืดหยุ่นซึ่งสามารถเปลี่ยนเส้นทางการทำคะแนนของคุณได้อย่างสิ้นเชิง

การอัปเกรดสมบัติเป็นสิ่งสำคัญสำหรับการก้าวหน้าในระยะยาว เกมมีระบบที่ตรงไปตรงมาซึ่งใช้การสุ่มได้สมบัติซ้ำและเหรียญเพื่อเพิ่มระดับของสมบัติ การเพิ่มระดับแต่ละครั้งจะช่วยเพิ่มเอฟเฟกต์หลักของสมบัติ เช่น การเพิ่มคะแนนที่ได้รับจากเยลลี่เฉพาะอย่าง หรือการลดคูลดาวน์ของความสามารถแอคทีฟ

| คุณสมบัติ | รายละเอียด | ผลกระทบ |
| :--- | :--- | :--- |
| **การได้รับ** | ตั๋วสุ่มสมบัติชั้นเลิศ, คริสตัล, กิจกรรม | ปลดล็อกอุปกรณ์ใหม่และชิ้นส่วนซ้ำ |
| **การอัปเกรด** | ใช้เหรียญและวัสดุที่ซ้ำกัน | เพิ่มตัวคูณคะแนนและระยะเวลาการแสดงผล |
| **การทำงานร่วมกัน** | รวมกับคุกกี้และสัตว์เลี้ยงเฉพาะ | เปิดใช้งานโบนัสคอมโบพิเศษสำหรับการแข่งลีก |
| **ความต้องการพื้นที่จัดเก็บ** | ต้องการพื้นที่จัดเก็บอุปกรณ์ที่เสถียร (แนะนำ 3GB) | ป้องกันเกมค้างระหว่างการวิ่งที่มีความเข้มข้นสูง |

การระบุ **สมบัติที่ดีที่สุดใน คุกกี้รัน คลาสสิก** จะช่วยให้คุณประหยัดเหรียญนับล้านและเวลาในการฟาร์มหลายชั่วโมง เมต้าถูกแบ่งออกเป็นหมวดหมู่การเพิ่มคะแนน ประโยชน์ใช้สอย และการฟาร์มทรัพยากร สมบัติระดับ S เป็นตัวเลือกที่หลากหลายซึ่งยอดเยี่ยมในเกือบทุกการแข่งลีก ในขณะที่ตัวเลือกระดับ A จะโดดเด่นในการผสมผสานตัวละครเฉพาะหรือสภาพแวดล้อมการฟาร์ม ส่วนสมบัติระดับ B โดยทั่วไปจะเป็นตัวเลือกชั่วคราวสำหรับการก้าวหน้าในช่วงต้นเกม

| ระดับ | ชื่อสมบัติ | ประโยชน์หลัก | การจัดทีมที่แนะนำ |
| :--- | :--- | :--- | :--- |
| **S** | วงแหวนโล่ทอง | ดูดซับความเสียหายจากการชน, เพิ่มคะแนนเล็กน้อย | คุกกี้ภูตดอกบัว |
| **S** | แม่เหล็กเยลลี่ชั้นเลิศ | ดึงดูดเยลลี่ใกล้เคียงทั้งหมดโดยอัตโนมัติ | การวิ่งลีกคะแนนสูง |
| **A** | กล่องพลุเหรียญ | สร้างพลุเหรียญตามช่วงเวลาที่กำหนด | การตั้งค่าฟาร์มเหรียญ |
| **A** | แอมพูลฟื้นฟูศักดิ์สิทธิ์ | ฟื้นฟูพลังงานจำนวนเล็กน้อย | การเอาตัวรอดระยะไกล |
| **B** | รองเท้าวิ่งพื้นฐาน | เพิ่มความเร็วในการเคลื่อนที่พื้นฐานเล็กน้อย | การวิ่งเริ่มต้นและการก้าวหน้าช่วงแรก |

<CardGrid cols={3}>
  <Card title="วงแหวนโล่ทอง">
    - **ความสามารถในการอยู่รอดสูง**: ดูดซับการชนหนักเพื่อให้โมเมนตัมการวิ่งของคุณไม่หยุดชะงัก
    - **การป้องกันคะแนน**: ป้องกันการเสียคะแนนที่เกิดจากการชนสิ่งกีดขวาง
    - **เหมาะสำหรับมือใหม่**: แนะนำเป็นอย่างยิ่งสำหรับผู้เล่นที่ยังคงเรียนรู้รูปแบบการกระโดดและสไลด์ที่ยากลำบาก
  </Card>
  <Card title="แม่เหล็กเยลลี่ชั้นเลิศ">

    - **เส้นทางที่ปลอดภัย**: เก็บเยลลี่จากเส้นทางที่เสี่ยงโดยไม่จำเป็นต้องใช้ท่าทางที่อันตราย
    - **การทำงานร่วมกันแบบสากล**: ใช้งานได้ดีเป็นพิเศษกับคุกกี้และสัตว์เลี้ยงทุกรูปแบบ
  </Card>
  <Card title="กล่องพลุเหรียญ">
    - **การฟาร์มทรัพยากร**: สร้างเหรียญจำนวนมหาศาลในระหว่างการวิ่งทั่วไป
    - **แหล่งพลังงานสำหรับการอัปเกรด**: จำเป็นสำหรับการฟาร์มเหรียญที่จำเป็นในการอัปเกรดสมบัติระดับ S ของคุณ
    - **ประสิทธิภาพในการทำกิจกรรม**: เหมาะสำหรับการเคลียร์ภารกิจเก็บเหรียญอย่างรวดเร็ว
  </Card>
</CardGrid>

<Section icon={Swords} title="การจัดทีมที่ดีที่สุด: การจับคู่คุกกี้ สัตว์เลี้ยง และสมบัติ">

สมบัติจะดีได้เท่ากับคุกกี้และสัตว์เลี้ยงที่วิ่งอยู่ข้างๆ มัน เพื่อเพิ่มศักยภาพในการทำคะแนนของคุณให้สูงสุด คุณต้องสร้างทีมที่สอดประสานกัน ตัวอย่างเช่น การจับคู่สมบัติที่เน้นคะแนนกับตัวละครที่ทำคะแนนได้สูงอย่าง คุกกี้ภูตดอกบัว จะให้ผลลัพธ์ที่เหนือกว่ามากเมื่อเทียบกับการใช้การตั้งค่าที่ไม่เข้ากัน ด้านล่างนี้คือการจัดทีมที่มีประสิทธิภาพที่สุดสำหรับการเล่นแบบแข่งขันและการฟาร์มทรัพยากร

| ประเภทการจัดทีม | คุกกี้ | สัตว์เลี้ยง | สมบัติที่แนะนำ |
| :--- | :--- | :--- | :--- |
| **คะแนนลีกเมต้า** | ภูตดอกบัว | ผู้ดูแลดอกบัว | วงแหวนโล่ทอง, แม่เหล็กเยลลี่ชั้นเลิศ |
| **คะแนนสูงนักเดินทาง** | คุกกี้หลวงพี่สน | คู่หูสน | แอมพูลฟื้นฟูศักดิ์สิทธิ์, แม่เหล็กเยลลี่ชั้นเลิศ |
| **ฟาร์มเหรียญ** | คุกกี้รสนักเล่นแร่แปรธาตุ | ตาชั่งเหรียญ | กล่องพลุเหรียญ, กุญแจคลังทองคำ |
| **เริ่มต้นสำหรับมือใหม่** | คุกกี้รสผู้กล้า | ช็อกโก้ดรอป | รองเท้าวิ่งพื้นฐาน, วงแหวนโล่ทอง |

การตั้งค่าแต่ละอย่างเหล่านี้ตอบโจทย์สไตล์การเล่นที่เฉพาะเจาะจง การจัดทีมคะแนนลีกเมต้าเน้นหนักไปที่การเก็บเยลลี่มูลค่าสูงทุกชิ้นที่สร้างโดย คุกกี้ภูตดอกบัว โดยใช้แม่เหล็กเยลลี่ชั้นเลิศเพื่อให้แน่ใจว่าจะไม่มีคะแนนถูกทิ้งไว้เบื้องหลัง ในขณะเดียวกัน การจัดทีมฟาร์มเหรียญจะให้ความสำคัญกับการสร้างความมั่งคั่ง โดยเปลี่ยนเส้นทางการวิ่งมาตรฐานให้เป็นเส้นทางเก็บเหรียญที่ทำกำไร

<Section icon={Target} title="คู่มือการอัปเกรดและการเลือกสมบัติทีละขั้นตอน">

เพื่อเพิ่มประสิทธิภาพการก้าวหน้าของบัญชีของคุณโดยไม่เสียทรัพยากรอันมีค่า โปรดปฏิบัติตามแผนงานเชิงกลยุทธ์นี้ คู่มือทีละขั้นตอนนี้ออกแบบมาเพื่อช่วยให้คุณเปลี่ยนจากนักวิ่งมือใหม่เป็นผู้เล่นที่ทำคะแนนสูงในการแข่งขันโหมดลีกแบบเรียลไทม์

  </Step>
