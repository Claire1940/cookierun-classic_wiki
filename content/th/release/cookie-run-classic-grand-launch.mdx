การเปิดตัวอย่างเป็นทางการของ **คุกกี้รัน คลาสสิก** ในวันที่ 25 มิถุนายน 2026 ถือเป็นการกลับมาของเกมวิ่งมุมมองด้านข้างที่เป็นที่รักของทุกคนสู่สโตร์ทั่วโลก [1] พัฒนาและเปิดให้บริการโดย Devsisters ตัวเกมในเวอร์ชันนี้เน้นไปที่กลไกสุดคลาสสิกอย่างการกระโดด การสไลด์ และการหลบหลีกสิ่งกีดขวาง พร้อมกับเก็บเยลลี่แสนอร่อยเพื่อหลบหนีออกจากเตาอบของแม่มด [1] แตกต่างจากภาคแยกหรือการดัดแปลงใน Roblox นี่คือเกมมือถือตัวเต็มที่สร้างขึ้นใหม่ทั้งหมดสำหรับแพลตฟอร์ม iOS, iPadOS และ Android [1]

เพื่อให้มั่นใจว่าจะได้รับประสบการณ์การเล่นเกมที่ราบรื่นตั้งแต่วันแรก ผู้เล่นควรตรวจสอบว่าอุปกรณ์มือถือของตนมีคุณสมบัติตรงตามความต้องการขั้นต่ำและขั้นแนะนำของระบบ ด้านล่างนี้คือรายละเอียดข้อกำหนดอย่างเป็นทางการของแต่ละแพลตฟอร์ม [1]

<Section icon={Target} title="การพัฒนาสำหรับผู้เริ่มต้น: กลยุทธ์การวิ่งแบบทีละขั้นตอน">

การเริ่มต้นเล่นในช่วงแรกอาจทำให้รู้สึกสับสนได้เนื่องจากมีกิจกรรมและตัวเลือกการอัปเกรดมากมายเปิดใช้งานพร้อมกัน การจัดระเบียบขั้นตอนการเล่นในช่วงเริ่มต้นจะช่วยให้คุณใช้ทรัพยากรได้อย่างคุ้มค่าที่สุดและหลีกเลี่ยงข้อผิดพลาดทั่วไปในการพัฒนาตัวละคร [3]

รูปแบบการเล่นหลักของเกมขึ้นอยู่กับจังหวะที่แม่นยำและการตอบสนองที่รวดเร็ว [3] เนื่องจากพลังชีวิตของคุณจะลดลงเรื่อยๆ ในขณะที่วิ่ง การหลีกเลี่ยงการชนจึงเป็นปัจจัยที่สำคัญที่สุดในการทำคะแนนให้ได้สูง [3] การชนสิ่งกีดขวางไม่เพียงแต่ทำให้เสียพลังชีวิตจำนวนมาก แต่ยังทำให้ความเร็วของคุณหยุดชะงักชั่วคราว ส่งผลให้พลาดเยลลี่ที่มีค่าไป [1, 3]

  </Step>

<Section icon={Trophy} title="รางวัลฉลองเปิดตัวและการแลกคูปอง DevPlay">

เพื่อเฉลิมฉลองการเปิดตัวทั่วโลก ได้มีการเปิดตัวกิจกรรมและรางวัลโปรโมชันต่างๆ ในช่วงเปิดตัว [1, 3] รางวัลเหล่านี้ได้รับการออกแบบมาเพื่อให้ผู้เล่นใหม่เริ่มต้นได้อย่างก้าวกระโดดในการสะสมตัวละครและอัปเกรดอุปกรณ์ของตนเอง [1, 3]

การทำความเข้าใจว่าต้องรับรางวัลเหล่านี้ที่ไหนและอย่างไรเป็นสิ่งสำคัญ [3] แม้ว่าของขวัญบางอย่างจะถูกส่งไปยังกล่องจดหมายในเกมโดยตรง แต่ของขวัญอื่นๆ อาจต้องเข้าร่วมกิจกรรมเปิดตัวเฉพาะ หรือใช้ระบบคูปองของ DevPlay ด้วยตนเอง [1, 3]

| ชื่อรางวัล | สิ่งของภายใน | สถานที่รับรางวัล | หมายเหตุวันหมดอายุ |
| --- | --- | --- | --- |
| แพ็กเกจของขวัญฉลองเปิดตัว | 300 คริสตัล, 100,000 เหรียญ, ตั๋วสุ่มสมบัติสูงสุด 3 ใบ | กล่องจดหมายในเกม | รับสิทธิ์ในช่วงระยะเวลาเปิดตัว |

| บัฟสำหรับผู้เริ่มต้น | ตัวคูณเพิ่มค่าประสบการณ์และเหรียญชั่วคราว | บัฟที่เปิดใช้งานอยู่ | เปิดใช้งานโดยอัตโนมัติสำหรับบัญชีใหม่ |

<Callout type="success" title="แลกรับรางวัลสำเร็จ">
หากต้องการแลกรับโค้ดในอนาคต ให้ไปที่หน้า [แลกคูปอง DevPlay](https://coupon.devplay.com/coupon/crg/en) อย่างเป็นทางการ กรอก Player ID ของคุณ (พบได้ในการตั้งค่าเกม) และป้อนโค้ดที่ใช้งานได้ตามที่ระบุไว้ทุกประการ [3] รีสตาร์ทเกมของคุณเพื่อรับรางวัลในกล่องจดหมาย [3]

Player ID ของคุณคือรหัสตัวอักษรและตัวเลขเฉพาะที่กำหนดให้กับบัญชีของคุณ [3] วิธีค้นหา ให้แตะไอคอนฟันเฟืองที่มุมขวาบนของล็อบบี้หลัก ไปที่แท็บบัญชี (Account) และคัดลอก ID ไปยังคลิปบอร์ดของคุณ [3] ขอแนะนำเป็นอย่างยิ่งให้บันทึก ID นี้ไว้ เนื่องจากจำเป็นต้องใช้ทั้งสำหรับการแลกคูปองและการติดต่อฝ่ายบริการลูกค้า [3]

</Section>

<Section icon={Swords} title="เทียร์ลิสต์: การจัดผลัด (Combi) คุกกี้ เพท และสมบัติที่ดีที่สุด">

การทำคะแนนให้ได้เปรียบในการแข่งขันโหมดลีก (League) แบบเรียลไทม์ จำเป็นต้องมีความเข้าใจอย่างลึกซึ้งเกี่ยวกับการทำงานร่วมกัน (Synergy) ของตัวละคร [1, 3] การติดตั้งยูนิตที่มีระดับความหายากสูงสุดเพียงอย่างเดียวจะไม่ให้ผลลัพธ์ที่ดีที่สุด หากทักษะของพวกมันไม่เกื้อหนุนกัน [1, 3] การผสมผสานที่ลงตัวระหว่างคุกกี้ เพท และสมบัติที่ได้รับการอัปเกรดคือเคล็ดลับในการครองอันดับบนลีดเดอร์บอร์ด [1, 3]

เพื่อช่วยให้คุณลงทุนทรัพยากรที่หามาได้อย่างคุ้มค่า เราได้จัดหมวดหมู่รายชื่อตัวละครที่มีอยู่ในช่วงเปิดตัวนี้ออกเป็นเทียร์ต่างๆ ตามศักยภาพในการทำคะแนน ความง่ายในการใช้งาน และความหลากหลายในการใช้งานโดยรวม [3]

ความปลอดภัยของบัญชีควรเป็นสิ่งที่คุณให้ความสำคัญสูงสุด [3] การเล่นด้วยบัญชีเกสต์จะทำให้คุณเสี่ยงต่อการสูญเสียข้อมูลความคืบหน้าอย่างถาวร หากตัวเกมเกิดค้าง อัปเดตผิดพลาด หรือถูกถอนการติดตั้งโดยไม่ได้ตั้งใจ [3]

</Callout>
