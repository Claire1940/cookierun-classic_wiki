ในช่วงเปิดตัว ตัวเกมมีคุกกี้ที่แฟนๆ ชื่นชอบและคุกกี้สุดพิเศษในธีม "K-Culture" ที่ออกแบบมาสำหรับเวอร์ชันนี้โดยเฉพาะ การอัปเดตปัจจุบันในชื่อ "Grand Launch" ได้เน้นย้ำถึงตัวละครระดับสูงหลายตัวซึ่งเหมาะสำหรับทั้งผู้เริ่มต้นและผู้เล่นสายแข่งขัน

**ตัวละครแนะนำช่วงเปิดตัว:**
- **คุกกี้ภูตรากบัว**: ตัววิ่งทำคะแนนสูงที่โดดเด่นในกิจกรรมเปิดตัว
- **คุกกี้พระสน**: ส่วนหนึ่งของการอัปเดต "ปาร์ตี้ท่องเที่ยว" ที่มาพร้อมทักษะการเคลื่อนไหวที่ไม่เหมือนใคร
- **คุกกี้พิเศษ K-Culture**: คุกกี้พิเศษที่ได้รับแรงบันดาลใจจากอาหารเกาหลี มีเฉพาะในเวอร์ชันคลาสสิกเท่านั้น

| ชื่อคุกกี้ | ประเภท | สัตว์เลี้ยงที่แนะนำ | เหมาะสำหรับ |
| :--- | :--- | :--- | :--- |
| **คุกกี้ภูตรากบัว** | ทำคะแนน | บัวผู้พิทักษ์ | ลีกคะแนนสูง |
