<Section icon={Target} title="ระบบการเล่นหลักและกลยุทธ์สำหรับผู้เล่นเริ่มต้น">

ระบบการเล่นหลังจาก **วันเปิดให้บริการทั่วโลกของ คุกกี้รัน คลาสสิก เวอร์ชันมือถือ 1.0** ยังคงรักษาสูตรสำเร็จดั้งเดิมเอาไว้ เป้าหมายหลักของคุณคือการวิ่งไปให้ไกลที่สุดเท่าที่จะทำได้ในขณะที่เก็บเยลลี่และหลบหลีกอุปสรรคของแม่มด เวอร์ชัน "คลาสสิก" นี้เน้นความแม่นยำมากกว่าการกดใช้ความสามารถที่ซับซ้อน ทำให้ง่ายต่อการเริ่มต้นเล่น แต่ยากที่จะเชี่ยวชาญในระดับลีกที่สูงขึ้น

| ระบบการเล่น | คำอธิบาย | เคล็ดลับกลยุทธ์ |
| :--- | :--- | :--- |
| **กระโดด** | หลบสิ่งกีดขวางบนพื้นและช่องว่าง | แตะสองครั้งเพื่อกระโดดสูงขึ้นข้ามลังไม้ที่ซ้อนกัน |
| **สไลด์** | ลอดใต้สิ่งกีดขวางที่ห้อยอยู่ | กดค้างไว้เพื่อก้มต่ำ และปล่อยอย่างรวดเร็วเพื่อวิ่งต่อ |
| **เยลลี่** | แหล่งคะแนนหลักของเกม | วิ่งตามเส้นทางของเยลลี่เพื่อเส้นทางที่ปลอดภัยที่สุดในแต่ละด่าน |
| **พลังชีวิต (พลังงาน)** | ลดลงตามเวลาหรือเมื่อชนสิ่งกีดขวาง | ให้ความสำคัญกับ **ขวดยาเพิ่มพลังชีวิต** มากกว่าเยลลี่หายากเพื่อวิ่งได้ไกลขึ้น |
| **โบนัสคอมบิ** | การทำงานร่วมกันระหว่างคุกกี้และเพ็ท | จับคู่เพ็ทเฉพาะกับคุกกี้ที่เป็นคู่หูเพื่อรับคะแนนพิเศษ |

<Callout type="info" title="การแข่งขันในลีก">
การเล่นในลีกแบบเรียลไทม์คือหัวใจหลักของการแข่งขัน คุณจะถูกจับคู่กับผู้เล่นที่มีระดับฝีมือใกล้เคียงกันทั่วโลกเพื่อแข่งขันทำคะแนนสูงสุดในด่านที่จะหมุนเวียนทุกสัปดาห์
</Callout>
