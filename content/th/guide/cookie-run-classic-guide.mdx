- **คู่มือ คุกกี้รัน คลาสสิก**: เน้นไปที่วงจรหลักของการกระโดด สไลด์ และการเก็บเยลลี่เพื่อหนีจากเตาอบของแม่มด
- **รางวัลเปิดตัว**: ผู้เล่นทุกคนสามารถรับ **300 คริสตัล, 100,000 เหรียญ** และ **ตั๋วสุ่มสมบัติระดับสูงสุด 3 ใบ** ในช่วงกิจกรรมเปิดตัวอย่างเป็นทางการ
- **กลยุทธ์ที่ดีที่สุด**: ให้ความสำคัญกับการอัปเกรด **คอมบคุกกี้-เพ็ท-สมบัติ** เพียงชุดเดียว แทนที่จะกระจายทรัพยากรไปทั่วตัวละครทั้งหมด

- **โหมดลีก**: แข่งขันทำคะแนนแบบเรียลไทม์เพื่อไต่อันดับกระดานผู้นำระดับโลกและรับรางวัลซีซันสุดพิเศษ

<Section icon={Target} title="โค้ด คุกกี้รัน คลาสสิก & คู่มือการเติมโค้ด">

การเติมโค้ดใน **คุกกี้รัน คลาสสิก** จะดำเนินการผ่านพอร์ทัลภายนอกของ DevPlay สิ่งนี้ช่วยให้แน่ใจว่ารางวัลจะซิงค์กับบัญชีของคุณไม่ว่าจะใช้อุปกรณ์ใดก็ตาม แม้ว่าจะมีการปล่อยรหัสคูปองสาธารณะเป็นระยะๆ ผ่านช่องทางโซเชียลหลักอย่าง X (ชื่อเดิม Twitter) และ Instagram แต่ผู้เล่นต้องทำตามลำดับขั้นตอนที่กำหนดเพื่อให้แน่ใจว่าไอเทมจะถูกส่งไปยังกล่องจดหมายในเกม

**ไฮไลท์วิดีโอ:**
- ขั้นตอนการใช้งานเว็บไซต์เติมโค้ด DevPlay แบบทีละขั้นตอน
- วิธีค้นหา Player ID ในการตั้งค่าเกม
- การยืนยันรางวัลของขวัญเปิดตัว

<Steps>
  <Step num="1" title="หา Player ID ของคุณ">
    เปิด คุกกี้รัน คลาสสิก แล้วไปที่ **Settings (ตั้งค่า) > Game Info (ข้อมูลเกม)** คัดลอกที่อยู่อีเมล DevPlay หรือรหัส Player ID ที่แสดงอยู่ที่นั่น
  </Step>
  <Step num="2" title="เข้าสู่เว็บไซต์คูปอง">
    ไปที่ [หน้าคูปอง DevPlay](https://coupon.devplay.com/coupon/crg/en) อย่างเป็นทางการ นี่เป็นเว็บไซต์เดียวที่ได้รับอนุญาตสำหรับรางวัล คุกกี้รัน คลาสสิก
  </Step>
  <Step num="3" title="กรอกรายละเอียด">
    วาง Player ID ของคุณในช่องแรกและรหัสคูปองที่ใช้งานได้ในช่องที่สอง ตรวจสอบตัวอักษรที่พิมพ์ผิดหรือช่องว่างส่วนเกินให้ดี
  </Step>
  <Step num="4" title="รีสตาร์ทแอป">
    หลังจากคลิก "Redeem" ให้ปิดเกมให้สนิทแล้วเปิดใหม่อีกครั้ง รางวัลของคุณจะปรากฏใน **กล่องจดหมายในเกม** ภายในไม่กี่นาที
  </Step>
</Steps>

<Section icon={Map} title="กลไกการวิ่งเบื้องต้นและการควบคุม">

วงจรหลักของ **คุกกี้รัน คลาสสิก** ถูกออกแบบมาให้เข้าถึงง่ายและแม่นยำ ด้วยปุ่มเพียงสองปุ่ม คือ **Jump (กระโดด)** และ **Slide (สไลด์)** ความสำเร็จขึ้นอยู่กับความสามารถของคุณในการอ่านภูมิประเทศและตอบสนองต่อความเร็วของการวิ่ง เมื่อคุณก้าวหน้าขึ้น ความเร็วจะเพิ่มขึ้น ทำให้ต้องใช้ปฏิกิริยาตอบโต้ที่เร็วขึ้นเพื่อเก็บเยลลี่ในขณะที่หลีกเลี่ยงสิ่งกีดขวางที่ลดพลังชีวิตของคุกกี้

ผู้เล่นใหม่ควรมุ่งเน้นไปที่เส้นทางหลบหนีใน **"เตาอบของแม่มด"** ด่านแรกๆ เหล่านี้ถูกออกแบบมาเพื่อสอนจังหวะของเกม ให้ความสนใจอย่างใกล้ชิดกับการวางตำแหน่งของเยลลี่ยักษ์และไอเทมเยลลี่ความเร็วสูง เพราะสิ่งเหล่านี้จะมอบความเป็นอมตะชั่วคราวและเพิ่มคะแนนอย่างมหาศาล

| ประเภทเยลลี่ | ผลลัพธ์ | ลำดับความสำคัญของคะแนน |
| :--- | :--- | :--- |
| **เยลลี่พื้นฐาน** | เพิ่มคะแนนเล็กน้อย | ต่ำ |
| **เยลลี่ตัวอักษร** | เก็บให้ครบเพื่อเข้าสู่ "BONUSTIME" | สูง |
| **เยลลี่ความเร็วสูง** | ความเร็วสูง + เป็นอมตะ | สูงมาก |
| **เยลลี่ยักษ์** | คุกกี้ตัวขยายใหญ่ + ทำลายสิ่งกีดขวาง | สูง |

<Callout type="tip" title="เคล็ดลับการควบคุมระดับโปร">

</Callout>

  </Step>
  <Step num="3" title="ซิงค์ข้อมูล">
