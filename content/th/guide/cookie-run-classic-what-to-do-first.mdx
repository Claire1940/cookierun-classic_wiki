ช่วงเปิดตัวนี้จะเต็มไปด้วยกิจกรรม Grand Open, บัฟสำหรับผู้เริ่มต้น และรางวัลการล็อกอินรายวัน แทนที่จะกระโดดเข้าสู่การวิ่งแข่งขันด้วยตัวละครเริ่มต้น การสละเวลาไม่กี่นาทีเพื่อรับและจัดระเบียบไอเทมในคลังของคุณจะช่วยสร้างรากฐานที่มั่นคงสำหรับการพัฒนาของคุณ

| :--- | :--- | :--- |
| **คริสตัล** | 300 คริสตัล | รับจากแท็บกิจกรรมเปิดตัวในเกม |

| **ตั๋วสุ่มสมบัติระดับสูงสุด** | 3 ใบ | รับจากแท็บกิจกรรมเปิดตัวในเกม |
| **โบนัสล็อกอิน** | แตกต่างกันไปในแต่ละวัน | ล็อกอินทุกวันในช่วงกิจกรรม Grand Open |

คุกกี้รัน คลาสสิก ใช้การควบคุมแบบสองปุ่มสุดคลาสสิก ได้แก่ กระโดด (Jump) และสไลด์ (Slide) แม้ว่าระบบการเล่นจะเข้าใจง่าย แต่การฝึกจังหวะการควบคุมให้เชี่ยวชาญก็เป็นสิ่งจำเป็นสำหรับการวิ่งทำคะแนนสูงและการเอาชีวิตรอดในเตาอบของแม่มด

พลังงานในการวิ่งของคุณจะลดลงอย่างต่อเนื่องตามเวลา และการชนกับสิ่งกีดขวางจะทำให้พลังงานลดลงอย่างมากในทันที เพื่อเพิ่มระยะทางการวิ่งและเก็บเยลลี่ให้ได้มากที่สุด คุณต้องให้ความสำคัญกับการเอาชีวิตรอดมากกว่าการโลภเก็บไอเทม

<Steps>
  <Step num="1" title="ติดตั้งเกมอย่างเป็นทางการ">

  </Step>
  <Step num="2" title="เรียนรู้การควบคุมแบบสองปุ่ม">

  </Step>
  <Step num="3" title="ให้ความสำคัญกับเยลลี่และการเอาชีวิตรอด">

  </Step>
  <Step num="4" title="เปิดใช้งานบัฟสำหรับผู้เริ่มต้น">
    ใช้ประโยชน์จากบัฟพิเศษสำหรับผู้เริ่มต้นและภารกิจรายวันที่มีให้ในช่วงกิจกรรมเปิดตัว Grand Open
  </Step>
</Steps>

### คู่หูแนะนำสำหรับผู้เริ่มต้น

เมื่อเริ่มต้นเล่น ให้พยายามเน้นไปที่คู่หูเหล่านี้เพื่อสร้างทีมวิ่งที่เสถียร:

| ตัววิ่งคุกกี้ | เพทที่เข้าคู่ | จุดเด่นของคอมบิ |
| :--- | :--- | :--- |
| **คุกกี้พระสนไม้** | คู่หูกิจกรรม | เหมาะอย่างยิ่งสำหรับการทำภารกิจกิจกรรมปาร์ตี้ท่องเที่ยวให้สำเร็จ |
| **คุกกี้ร่างทรงรากบัว** | **ผู้ดูแลดอกบัว** | คอมบิเปิดตัวที่โดดเด่นพร้อมโอกาสในการทำคะแนนสูง |
| **คุกกี้วัฒนธรรม K** | คู่หูที่เข้าคู่ | ตัววิ่งเฉพาะเวอร์ชันพร้อมบัฟทำคะแนนพิเศษ |

### ลำดับความสำคัญในการอัปเกรดสมบัติ

สมบัติจะมอบโบนัสติดตัวที่สามารถส่งผลต่อการวิ่งทำคะแนนสูงสุดอย่างมาก ใช้ตั๋วสุ่มสมบัติระดับสูงสุดของคุณอย่างชาญฉลาด:

| ชื่อสมบัติ | เอฟเฟกต์หลัก | ลำดับความสำคัญในการอัปเกรด |
| :--- | :--- | :--- |
| **สมบัติระดับสูงสุด** | การสร้างคะแนนสูง | **สูง** - อัปเกรดทันทีสำหรับการวิ่งในลีก |
| **สมบัติอรรถประโยชน์** | การรักษาพลังงานหรือแม่เหล็กดูดเหรียญ | **ปานกลาง** - อัปเกรดเมื่อชุดทำคะแนนหลักมีความเสถียรแล้ว |
| **สมบัติกิจกรรม** | ตัวคูณคะแนนเฉพาะกิจกรรม | **ต่ำ** - อัปเกรดเฉพาะเมื่อภารกิจที่กำลังดำเนินอยู่ต้องการเท่านั้น |

<Callout type="tip" title="เน้นการอัปเกรดของคุณ">
หลีกเลี่ยงการกระจายทรัพยากรของคุณไปกับตัวละครหลายตัวเกินไป เน้นอัปเกรดคอมบิหลักเพียงหนึ่งชุดก่อนเพื่อไต่เต้าอันดับในลีก
</Callout>

</Section>

| :--- | :--- | :--- | :--- |
| **iOS / iPadOS** | iOS 15.0 ขึ้นไป | **2 ถึง 3 GB** | อัปเดต iOS, ปิดแอปพลิเคชันเบื้องหลัง และรีสตาร์ทอุปกรณ์ |
| **Android** | AOS 8.0 ขึ้นไป | **4 GB** | เคลียร์หน่วยความจำอุปกรณ์, อัปเดตระบบปฏิบัติการ Android และรีสตาร์ท |
