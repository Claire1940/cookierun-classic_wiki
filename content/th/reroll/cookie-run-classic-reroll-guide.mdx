การเชื่อมต่อบัญชียังช่วยให้คุณเล่นข้ามอุปกรณ์ได้ หากคุณรีโรลบนโปรแกรมจำลอง Android แต่ต้องการเล่นบน iPhone บัญชี DevPlay จะเป็นสะพานเชื่อมที่นำคุกกี้ของคุณตามไปด้วย

| วิธีการเชื่อมต่อ | ข้อดี | ข้อเสีย |
| :--- | :--- | :--- |
| **อีเมล DevPlay** | เข้าถึงได้จากทุกที่ กู้คืนได้ง่าย | ต้องจำรหัสผ่านแยกต่างหาก |
| **Google/Apple ID** | ล็อกอินได้เพียงแตะครั้งเดียว ปลอดภัยมาก | อาจเปลี่ยนแพลตฟอร์ม OS ได้ยาก |
| **บัญชี Guest** | รวดเร็วที่สุดสำหรับการรีโรล | **ความเสี่ยงสูง** ที่ข้อมูลจะสูญหายทั้งหมด |

<Callout type="warning" title="หมายเหตุเกี่ยวกับการแลกรางวัล">
หลังจากเชื่อมต่อแล้ว อย่าลืมไปที่หน้าคูปอง DevPlay อย่างเป็นทางการเพื่อตรวจสอบรหัสที่ใช้งานได้ แม้จะยังไม่มีการยืนยันรหัสในช่วงเปิดตัวทันที แต่รหัสใหม่มักจะถูกโพสต์บนบัญชี X อย่างเป็นทางการ
