### 보물 및 업그레이드 우선순위

보물은 달리기의 양상을 완전히 바꿀 수 있는 패시브 장비 역할을 합니다. 레벨이 높은 보물일수록 더 강력한 점수 배율과 유틸리티 효과를 제공하므로 보물 업그레이드는 매우 중요합니다.

| 보물 이름 | 패시브 효과 | 업그레이드 우선순위 | 점수 팁 |
| :--- | :--- | :--- | :--- |
| **최고급 골든 실드** | 경미한 장애물 충돌을 흡수하고 자석 효과 부여 | **매우 높음** | 달리기를 이어가고 놓친 젤리를 수집하는 데 필수적 |
| **젤리 캔디 케인** | 일정 간격으로 고가치의 사탕 젤리 추가 생성 | **높음** | 일반 질주 구간에서 기본 점수를 크게 증폭 |
| **마녀의 부츠** | 슬라이드 속도를 높이고 슬라이드 중 체력 감소 감소 | **보통** | 천장이 낮은 기술적인 맵에서 탁월한 효율 발휘 |
| **은화 왕관** | 획득한 은화 코인을 일정 비율로 승수 증가 | **낮음** | 철저히 코인 파밍용으로만 사용, 리그전에서는 사용 금지 |

쾌적한 게임 플레이를 위해서는 모바일 기기가 최소 하드웨어 요구 사양을 충족하는지 확인하는 것이 중요합니다. 오래된 운영체제나 저장 공간이 부족한 기기에서 게임을 실행하면 성능 저하, 프레임 드랍 또는 예상치 못한 앱 종료 현상이 발생할 수 있습니다.

### 기기 사양 및 최소 요구 사양

| 플랫폼 | 최소 OS 버전 | 권장 RAM | 필수 여유 저장 공간 |
| :--- | :--- | :--- | :--- |
| **iOS / iPadOS** | iOS 15.0 이상 | 3 GB RAM 이상 | 3 GB 이상의 여유 공간 |
| **Android** | AOS 8.0 이상 | 4 GB RAM 이상 | 4 GB 이상의 여유 공간 |

기술적인 문제가 발생할 경우, 공식 고객센터에 문의하기 전에 아래의 문제 해결 가이드를 참고하여 직접 해결을 시도해 보세요.

### 문제 해결 빠른 팁

| 문제 현상 | 예상 원인 | 해결 방법 | 지원 상태 |
| :--- | :--- | :--- | :--- |
| **실행 시 앱 종료** | 기기 메모리 부족 또는 오래된 운영체제 | 백그라운드 앱 종료, 기기 재부팅 및 OS 업데이트 | 대부분의 경우 기본 단계로 해결 가능 |
| **앱 업데이트 불가** | 기기 내 여유 저장 공간 3 GB 미만 | 사용하지 않는 사진, 동영상 또는 앱을 삭제하여 공간 확보 | 흔히 발생하는 문제로 저장 공간 설정 확인 필요 |
| **네트워크 불안정 / 끊김** | 와이파이 연결 불량 또는 셀룰러 데이터 신호 저하 | 안정적인 초고속 네트워크로 전환하거나 라우터 재부팅 | 네트워크 환경에 따른 현상으로 앱 오류가 아님 |
| **게스트 진행 데이터 유실** | 계정 연동 없이 앱을 삭제했거나 기기를 초기화함 | 초기화 전에 DevPlay, Google 또는 Apple ID에 계정 연동 | 연동되지 않은 계정은 복구가 불가능함 |

게임 플레이를 원활하게 유지하고 소중한 데이터를 안전하게 보호하기 위해 아래의 일일 체크리스트를 기억해 두세요.
