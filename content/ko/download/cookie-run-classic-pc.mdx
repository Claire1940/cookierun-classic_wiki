| 플랫폼 | 추천 대상 | 주요 장점 |
| :--- | :--- | :--- |

### 보물 업그레이드 우선순위
보물은 장기적인 고득점의 핵심입니다. 다음 업데이트에서 대체될 수 있는 쿠키와 달리, 고품질 보물은 메타가 여러 번 바뀌어도 유용하게 유지됩니다.

| 보물 종류 | 효과 | 업그레이드 우선순위 |
| :--- | :--- | :--- |
| **자력 장비** | 멀리 있는 젤리를 끌어당김 | **높음** (PC 필수) |
| **보호막/부활** | 장애물 충돌로 인한 사망 방지 | **보통** (초보자 추천) |
| **점수 배율 증가** | 특정 젤리 점수 증가 | **높음** (후반 리그용) |
| **코인 부스터** | 골드 획득량 증가 | **낮음** (파밍 전용) |

**쿠키런 클래식 PC**의 핵심은 **점프와 슬라이드**라는 두 버튼 시스템입니다. 단순해 보이지만, 마녀의 오븐 스테이지 디자인은 갈수록 복잡해지므로 젤리 획득을 극대화하려면 프레임 단위의 정확한 입력이 필요합니다.

- **2단 점프**: 점프 버튼을 두 번 눌러 더 높은 발판에 도달하거나 넓은 구덩이를 건너뛰세요.
- **슬라이드 유지**: 슬라이드 버튼을 길게 눌러 낮게 매달린 장애물 아래를 지나가세요. PC에서는 터치 제스처보다 매핑된 키를 누르고 있는 것이 훨씬 더 정확합니다.
- **젤리 패턴**: 젤리는 종종 가장 안전한 경로로 인도하는 '길'을 형성합니다. 숨겨진 함정을 피하려면 젤리 길을 따라가세요.

<Callout type="info" title="리그 전략">

</Callout>

<Section icon={Shield} title="계정 이전 및 문제 해결">

다양한 기기에서 계정을 유지하는 것은 **쿠키런 클래식 PC** 플레이어들의 공통된 관심사입니다. 이 게임은 DevPlay를 사용하기 때문에, 크리스탈 하나도 잃지 않고 안드로이드 폰에서 PC 에뮬레이터로 전환할 수 있습니다.

| 문제 상황 | 해결 방법 |
| :--- | :--- |
| **게임 튕김 현상** | 3GB 이상의 여유 공간을 확보하고 기기를 재부팅하세요. |
| **연결 오류** | 와이파이(Wi-Fi)에서 데이터로 전환하거나 PC에서 안정적인 LAN 선을 사용하세요. |
| **업데이트 실패** | 앱 캐시를 지우거나 OS 업데이트를 확인하세요 (iOS 15+ / 안드로이드 8.0+). |
| **보상 누락** | 앱을 재시작하여 우편함 데이터를 새로고침하세요. |

<Callout type="warning" title="중요 경고">
계정을 구글, 애플 ID 또는 DevPlay에 연동하기 전에 **절대 앱을 삭제하지 마세요**. 게스트 계정은 기기에 로컬로 저장되므로, 게스트 상태에서 게임을 삭제하면 진행 상황이 영구적으로 소실됩니다.
</Callout>
