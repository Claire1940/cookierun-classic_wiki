<Section icon={Shield} title="アカウント移行＆トラブルシューティング">

異なるデバイス間でアカウントを維持することは、**クッキーランクラシック PC版**のプレイヤーにとって共通の関心事です。本作はDevPlayを使用しているため、クリスタルを1個も失うことなく、AndroidスマートフォンからPCエミュレータへ移行することができます。

| 発生している問題 | 考えられる解決策 |
| :--- | :--- |
| **ゲームの強制終了** | 3GB以上の空き容量を確保し、デバイスを再起動する |
| **接続エラー** | Wi-Fiからモバイルデータ通信に切り替えるか、PCで安定した有線LANを使用する |
| **アップデートの失敗** | アプリのキャッシュをクリアするか、OSのアップデートを確認する（iOS 15以上/AOS 8以上） |
| **報酬が届かない** | アプリを再起動してメールボックスのデータを更新する |

<Callout type="warning" title="重要な警告">
Google、Apple ID、またはDevPlayにアカウントを連携する前に、**絶対にアプリを削除しないでください**。ゲストアカウントはローカルに保存されているため、ゲストの状態でゲームをアンインストールすると、進行状況が完全に削除されます。
</Callout>

<Steps>
  <Step num="1" title="モバイル版での連携">
    設定 ＞ ゲーム情報 ＞ アカウント連携を開きます。DevPlayを選択し、有効なメールアドレスを入力します。
  </Step>
  <Step num="2" title="PCへの移行">
    PCにゲームをインストールします。タイトル画面で「ゲスト」ではなく「ログイン」を選択し、DevPlayのログイン情報を使用します。
  </Step>
  <Step num="3" title="確認">
    新しいプラットフォームでゲーム内購入を行う前に、レベルとクリスタル数が一致していることを確認してください。
  </Step>
</Steps>
