<Section icon={Sparkles} title="アカウント移行とトラブルシューティング">

アカウントを連携せずにデバイスを交換したりシステムをリセットしたりすると、ゲームの進行状況が失われることがよくあります。冒険を守るために、常にゲームの進行状況を永続的なログイン方法に紐付けてください。

クラッシュ、ロード画面のフリーズ、接続エラーなどの技術的な問題が発生した場合は、公式サポートチームが推奨する以下の標準的なトラブルシューティング手順に従ってください。

| 問題 | 主な原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **アプリのクラッシュ** | デバイスのメモリ不足 | バックグラウンドアプリを閉じ、デバイスを再起動 |
| **更新の失敗** | ストレージ容量の不足 | デバイスの空き容量を少なくとも3GB確保 |
| **接続エラー** | 不安定なネットワーク | モバイルデータ通信から安定したWi-Fiに切り替え |

<Callout type="warning" title="データ紛失の警告">
連携されていないゲストアカウントは、デバイスの一時キャッシュにローカル保存されます。アプリケーションのアンインストールやデバイスデータの消去を行うと進行状況が永久に失われ、未連携のアカウントはカスタマーサポートでも復旧できません。
</Callout>
