<Section icon={Target} title="初心者ガイド：ランをマスターする">

クッキーラン：クラシックの旅を始めるのは簡単ですが、リーダーボードの上位に入るには正確な操作が必要です。ゲームは複雑な入力よりもタイミングを重視した2ボタン操作が中心です。主な目的は、ゼリーを集めながら魔女のオーブンから逃げ切ることです。

<Steps>
  <Step num="1" title="基本の操作サイクル">
    左側の**ジャンプ**ボタンで穴や障害物を飛び越えます。右側の**スライド**ボタンで低い障害物をくぐり抜けます。ジャンプを2回タップするとより高く跳ぶことができ、高い場所にあるゼリーを取るために不可欠です。
  </Step>
  <Step num="2" title="ゼリーの優先順位">
    すべてのゼリーが同じ価値ではありません。**巨大ゼリー**や、BONUS TIMEの文字を完成させる**アルファベットゼリー**を優先しましょう。ボーナスタイムに入ることが、障害物のリスクなしにスコアを稼ぐ最も効果的な方法です。
  </Step>
  <Step num="3" title="体力の管理">
    クッキーには減少していく体力ゲージがあります。ステージ中に配置された**回復ポーション**を拾うことが、ランを長く続けるために非常に重要です。体力がゼロになると、「復活」効果を持つ宝物やペットの能力がない限り、ランは終了します。
  </Step>
</Steps>

<Callout type="tip" title="プロのテクニック">
「パニックジャンプ」は避けましょう。後半のステージでは、ジャンプを誘っておいて空中で2番目の障害物に当てるような罠が設計されています。ハイスコアを狙う前に、ステージのリズムを覚えましょう。
</Callout>

| 操作タイプ | アクション | 主な用途 |
| :--- | :--- | :--- |
| **シングルジャンプ** | ジャンプを1回タップ | 小さな穴や1つの障害物を越える |
| **2段ジャンプ** | ジャンプを2回タップ | 高い足場への移動や壁を越える |
| **スライド** | スライドを長押し | トンネルや低い天井を通り抜ける |
| **コンボジャンプ** | ジャンプ後にスライド | 特定のゼリー配置に合わせた特殊な動き |

<Section icon={Swords} title="リーグモードとスコア競争">

基本をマスターし、強力なコンビを組んだら、**リーグ**に挑戦しましょう。これは世界中のプレイヤーとリアルタイムで競い合うモードです。スコアは走行距離、集めたゼリー、そして装備のレベルによって決まります。

**スコア最適化戦略:**
- **正確な操作**: 障害物に当たるたびに速度と体力が減少します。「ノーミスボーナス」がゴールドランクとシルバーランクの分かれ目になることがよくあります。
- **宝物のレベルアップ**: 強化された宝物はスコア倍率を飛躍的に高めます。リリース記念の最高級宝物チケットは慎重に使いましょう。
- **イベントバフ**: 特定の時間帯にコインや経験値の獲得量が増える初心者向けバフが実施されていないか確認しましょう。

| リーグランク | 報酬ティア | 競争の激しさ |
| :--- | :--- | :--- |
| **ブロンズ / シルバー** | 基本素材 | カジュアル / 初心者 |
| **ゴールド / エメラルド** | クリスタル / 霊魂石 | 中級者 / アクティブ |
| **ルビー / ダイヤモンド** | レア宝物 / 称号 | 上級者 / メタ重視 |

<Callout type="warning" title="体力管理の重要性">
リーグモードでは、距離もゼリーと同じくらい重要です。リスクのあるゼリーの塊か、回復ポーションかの選択を迫られたら、**常に回復ポーションを選びましょう**。
</Callout>
