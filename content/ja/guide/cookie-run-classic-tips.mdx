<Section icon={Target} title="基本メカニクス：ジャンプとスライドの習得">

**クッキーラン：クラシック攻略** の核心は、シンプルな2ボタン操作を極めることにあります。序盤はタップするだけで簡単に進めるように見えますが、上位のリーグ階層に入ると難易度が急激に上昇します。正確な動きこそが、記録更新とゲームオーバーの分かれ目となります。

**プレイの注目ポイント:**
- 大きな隙間を飛び越えるための2段ジャンプの習得。
- 狭い通路でゼリー回収を最大化するためのスライドのタイミング。
- 不必要なダメージを避けるための障害物の当たり判定（ヒットボックス）の把握。

<Callout type="warning" title="「欲張り」なミスを避ける">
画面上のすべてのゼリーを回収しようとしてリズムを崩すプレイヤーが多く見られます。スコア倍率が途切れたり、体力が尽きたりするリスクを冒してまでゼリーを取るより、危険なゼリーはスキップするほうが賢明です。
</Callout>

<Steps>
  <Step num="1" title="マッスルメモリー（操作の慣れ）を構築する">
    2段ジャンプのタイミングを練習しましょう。後半のステージでは、安全に飛び越えるために1回目と2回目のタップの間に特定のディレイ（間隔）が必要な障害物が多く登場します。
  </Step>
  <Step num="2" title="スライドを極める">
    スライドは天井が低い場所を通るためだけのものではありません。横方向のポジションを微調整し、次のゼリーの列やジャンプに備えるためにも活用できます。
  </Step>
  <Step num="3" title="ルートを覚える">
    各ステージのレイアウトは固定されています。ゼリーと障害物のパターンに注意を払いましょう。次に何が来るかを知ることで、余裕を持ってジャンプの準備ができます。
  </Step>
</Steps>

<Section icon={Sparkles} title="宝物：最大ポイント獲得のための強化">

宝物は、走行を根本的に変えるパッシブまたはアクティブ装備として機能します。クッキーやペットとは異なり、宝物は汎用性が高いため、しっかり強化された宝物は多くの異なるコンビで使い回すことができます。

**最高級宝物チケット** を賢く使うことは、最も重要な攻略ポイントの一つです。「すべてのゼリー」のスコアを増加させる宝物や、ゼリーを引き寄せる磁力効果を持つ宝物を狙いましょう。これにより、リスクのある動きを減らすことができます。

| 宝物の種類 | 効果の説明 | 強化のメリット |
| :--- | :--- | :--- |
| **スコアブースター** | 特定のゼリーの得点を増加 | スコア倍率の上昇 |
| **磁力** | 近くのゼリーをクッキーに引き寄せる | 回収の効率化 / 安全性の向上 |
| **体力温存** | 体力の減少速度を抑える | 走行時間の延長 |
| **シールド** | 障害物への衝突を1回防ぐ | 生存率の向上 |

<Callout type="info" title="宝物の管理">
強化素材を分散させすぎないようにしましょう。自分のプレイスタイルに合った3つの主要な宝物のレベル上げに集中してください。レベル5の宝物1つは、レベル2の宝物3つよりもはるかに大きな影響を与えます。
</Callout>

<Section icon={Trophy} title="リーグ戦略：競争でのハイスコア獲得">

リーグモードは、世界中のプレイヤーを相手に自分の実力を試す場所です。このリアルタイム競争では、標準のアドベンチャーモードとは異なる考え方が求められます。リーグでは、一度のラッキーな走行よりも「安定性」が重要です。

**リーグ戦の重要戦術:**
1. **ルートの最適化**: すべてのリーグマップには、ゼリーが最も密集している「ゴールデンパス」が存在します。このパスを特定し、正確に通りましょう。
2. **バフの活用**: グランドオープン中に提供される初心者バフやイベントボーナスを活用して、ランキングを上げましょう。
3. **ミスの分析**: 障害物に当たった場合は、なぜそうなったのかを分析してください。スライドが遅かったのか、ジャンプのタイミングが悪かったのか？一つの繰り返されるミスを修正するだけで、ティアを一つ丸ごと上げることができます。

| リーグランク | 報酬ティア | 難易度レベル |
| :--- | :--- | :--- |
| **ブロンズ / シルバー** | 基本素材 | 初級 |
| **ゴールド / エメラルド** | クリスタル＆チケット | 中級 |
| **ルビー / ダイヤモンド** | 限定コスチューム | 上級 |
| **マスター / チャンピオン** | 最高級通貨 | エリート |

</Callout>

<Section icon={Shield} title="アカウント保護とトラブルシューティング">

技術的な問題やデータの紛失ほど、ハイスコアの連勝を台無しにするものはありません。クッキーラン：クラシックはライブサービスゲームであるため、アカウントが適切に連携されている場合にのみ、進行状況がDevsistersのサーバーに保存されます。

<Steps>
  <Step num="1" title="DevPlayに連携する">
    永続的に使用できるメールアドレスでDevPlayアカウントを作成しましょう。これがiOSとAndroidの間で進行状況を移行する最も信頼できる方法です。
  </Step>
  <Step num="2" title="ソーシャル連携を確認する">
    バックアップとして、Apple IDまたはGoogle Playアカウントも連携しておきましょう。万が一メールアドレスへのアクセスを失った際の予備の復旧手段になります。
  </Step>
  <Step num="3" title="ストレージを確認する">
    少なくとも3〜4GBの空き容量を確保してください。ゲームのアップデートファイルやキャッシュのために追加のスペースが必要です。
  </Step>
</Steps>

<Callout type="warning" title="クラッシュ防止プロトコル">
iOSやAndroidでクラッシュが発生する場合は、走行を開始する前にすべてのバックグラウンドアプリを閉じてください。高負荷なリーグ戦は、古いデバイスのRAMに負担をかける可能性があります。
</Callout>
