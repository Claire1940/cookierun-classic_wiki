<Section icon={Target} title="コアゲームプレイの仕組みと操作">

クッキーランクラシックでの成功は、2ボタンインターフェースの習熟度によって決まります。操作自体は簡単ですが、レベルデザインが進むにつれてゼリーや障害物の配置が複雑になり、正確な実行が求められます。すべてのランは「欲」（リスクのあるゼリーを狙う）と「生存」（体力を減らす衝突を避ける）のバランスです。

スコアリングシステムは多面的です。基本的なゼリーや特別なパワーアップを集めるだけでなく、選択したクッキーとペットの固有スキルによってポイントを獲得します。障害物は最大の敵です。衝突すると体力が減少するだけでなく、勢い（モーメンタム）も途切れてしまいます。これは競争の激しいリーグ戦において致命的となる可能性があります。

| 操作タイプ | アクション | スコアへの影響 |
| :--- | :--- | :--- |
| **シングルジャンプ** | ジャンプを1回タップして小さな隙間を越える | 勢いを維持し、空中のゼリーを回収する |
| **2段ジャンプ** | ジャンプを2回タップして高い足場に到達する | 隠しルートやボーナスタイムへの到達に不可欠 |
| **スライド** | スライドを長押しして低い障害物をくぐる | 狭い通路での生存に必要 |
| **スキル発動** | キャラクターのクールタイムに基づき自動発動 | 大幅なスコアアップや無敵状態を提供 |

<Section icon={Swords} title="初心者におすすめのクッキー・ペット・宝物">

強力な「コンビ」（組み合わせ）を構築することは、クッキーランクラシックのメタの核心です。コンビは、1体のクッキー、その相棒のペット、そして3つの宝物で構成されます。ゲームには限定のK-Cultureクッキーを含む多種多様なキャラクターが登場しますが、初心者は生存率が高い、または自動的にスコアを獲得できるユニットに焦点を当てるべきです。

2026年のリリース時のメタでは、新規プレイヤー向けにいくつかの際立った選択肢があります。「蓮華の精霊クッキー」とそのペット「ハスノハナ」のようなキャラクターは、ステージのレイアウトをまだ学習中のプレイヤーにとって優れた実用性を提供します。

| カテゴリ | 推奨の選択 | 高評価の理由 |
| :--- | :--- | :--- |
| **クッキー** | **蓮華の精霊クッキー** | 高いスコアポテンシャルと寛容なスキルタイミング |
| **クッキー** | **パインモンククッキー** | 長距離ランと生存に非常に優れている |
| **ペット** | **ハスノハナ** | 蓮華の精霊クッキーと完璧にシナジーを発揮する |
| **宝物** | **磁力オーラギア** | ゼリーを自動で引き寄せ、ジャンプの負担を軽減する |
| **宝物** | **シールドジェネレーター** | 障害物への衝突を1度防ぎ、ランを継続させる |

<Callout type="info" title="コンビシナジー">

<Section icon={Sparkles} title="リーグモードと高スコア戦略">

基本をマスターしたら、リアルタイムのリーグモードが競争の主な舞台となります。このモードでは、特定のマップで誰が最高スコアを達成できるか、世界中のプレイヤーと競い合います。マップは定期的に入れ替わるため、障害物の密度やゼリーの種類に合わせてコンビを適応させる必要があります。

リーダーボードの頂点に到達するには、単なる生存を超えなければなりません。ハイレベルなプレイには「欲」の管理が含まれます。どのゼリーがリスクを冒して2段ジャンプする価値があるか、完璧なランを維持するためにどれを無視すべきかを正確に知る必要があります。

| 戦略要素 | 重点エリア | プロのヒント |
| :--- | :--- | :--- |
| **ルートの最適化** | 障害物パターンの暗記 | 練習ランを使用して、最も安全なスライドのタイミングを見つける |
| **宝物のシナジー** | マップに合わせた装備 | 長く平坦なマップではスピードアップ系の宝物を使用する |
| **スキルタイミング** | ポイント獲得の最大化 | ゼリーが密集している区間でスキルを発動させるようにする |
| **リソース消費** | コアコンビの強化 | コインを複数のキャラに分散させず、1つのコンビの強化に集中する |

<Callout type="tip" title="練習あるのみ">
リーグランにはリソースが必要ですが、ほとんどのマップには「練習」オプションがあります。ランクを落としたり入場チケットを消費したりすることなく、レイアウトを学ぶために活用しましょう。
</Callout>

</Section>

<Section icon={Shield} title="アカウント管理とトラブルシューティング">

進行状況を保護することは、高スコアを出すことと同じくらい重要です。クッキーランクラシックをスムーズに動作させるには、安定したインターネット接続と特定のハードウェア要件が必要です。多くのプレイヤーがアップデート時やデバイスの切り替え時に問題に遭遇しますが、適切なアカウント連携を行うことでこれらを回避できます。

iOSやAndroidでクラッシュが発生する場合、公式サポートの推奨事項は、少なくとも3〜4GBの空きストレージを確保し、脱獄（ジェイルブレイク）や改造されたデバイスでのプレイを避けることです。

| 問題 | 解決策 |
| :--- | :--- |
| **進行状況の消失** | アンインストールする**前に**、DevPlay/Google/Appleに連携する |
| **インストールエラー** | 4GBのスペースを確保し、OSのバージョンを確認する |
| **接続失敗** | モバイルデータ通信から安定したWi-Fiに切り替える |
| **ゲームのクラッシュ** | バックグラウンドアプリを閉じ、デバイスを再起動する |
