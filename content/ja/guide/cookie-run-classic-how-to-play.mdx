<Section icon={Target} title="コアゲームプレイシステムのマスター">

**クッキーラン：クラシックの遊び方**の基本システムは、**「ジャンプ」**と**「スライド」**の2つのボタン操作を中心に展開します。操作自体はシンプルですが、ステージデザインには複雑なゼリーの配置や障害物のパターンが登場し、切り抜けるには正確なタイミングが求められます。主な目的は、体力ゲージが尽きる前にできるだけ遠くまで走り、同時に最高スコアを獲得することです。

<Steps>
  <Step num="1" title="ジャンプの仕組み">
    画面左側のジャンプボタンをタップして、穴や障害物を飛び越えます。2回タップすると「2段ジャンプ」になり、より高い足場に到達したり、より大きな障害物の塊を飛び越えたりできます。
  </Step>
  <Step num="2" title="スライドの仕組み">
    画面右側のスライドボタンを長押しして、低い位置にある障害物をくぐり抜けます。また、狭い通路に配置されたゼリーを回収するためにもスライドは不可欠です。
  </Step>
  <Step num="3" title="ゼリーの回収">
    ゼリーは主な得点源です。ゼリーの種類によって獲得できるスコアが異なります。「巨大化」や「高速ダッシュ」などの特殊アイテムを獲得すると、障害物を破壊しながら効率よくゼリーを回収できます。
  </Step>
  <Step num="4" title="体力の管理">
    クッキーの体力は時間経過とともに常に減少します。障害物に衝突すると、体力が大幅に減少してしまいます。マップ各所に散らばっている回復ポーションを回収して、走る距離を伸ばしましょう。
  </Step>
</Steps>

<Callout type="warning" title="障害物への注意">
障害物に当たると体力が減るだけでなく、走る勢い（モメンタム）も途切れてしまいます。競争の激しいリーグ戦では、たった1回の衝突がゴールドランクとプラチナランクを分ける決定打になることがあります。
</Callout>

| 操作タイプ | 主な用途 | 応用テクニック |
| :--- | :--- | :--- |
| **1段ジャンプ** | 小さな穴や低い障害物を飛び越える | 空中にあるゼリーの列に合わせたタイミング調整 |
| **2段ジャンプ** | 高い足場や広い穴を飛び越える | 滞空時間を伸ばして飛来物を避ける |
| **スライド** | 低い壁の下や狭い隙間をくぐる | 素早く指を離してジャンプへ移行する |
| **高速ダッシュ** | 無敵状態での高速ダッシュ | ルート上のすべてのゼリーを自動回収 |

</Section>

<Section icon={Sparkles} title="コンビシステム：クッキー、ペット、そして宝物">

**クッキーラン：クラシックの遊び方**において、走りの成果を左右するのは「コンビ」です。コンビは、1体のクッキー、1体のペット、そして最大3つの宝物で構成されます。それぞれの要素が互いに相乗効果（シナジー）を発揮し、スコア獲得、コイン稼ぎ、あるいは生存能力を最大化できるように組み合わせる必要があります。

<CardGrid cols={3}>
  <Card title="クッキー">
    走る主役です。それぞれのクッキーは、特殊なゼリーの生成や障害物の破壊など、独自の能力を持っています。現在の環境（メタ）では**レンコン妖怪味クッキー**がトップクラスの選択肢です。
  </Card>
  <Card title="ペット">
    頼れる相棒です。ペットは、回復ポーションの生成や磁力効果の発動など、パッシブバフやアクティブ効果を提供します。**ハス乳母**を対応するクッキーと組み合わせると、コンビボーナスを獲得できます。
  </Card>
  <Card title="宝物">
    装備品です。宝物は強力なグローバル効果を提供します。宝物はアップグレード可能で、リーグ戦における合計スコアに大きな影響を与えます。
  </Card>
</CardGrid>

宝物をアップグレードすることは、ハイスコアを安定して伸ばす最も確実な方法です。マップに依存する可能性のあるクッキーとは異なり、高レベルの宝物はほぼすべてのランにおいて価値を発揮します。

| 構成要素 | 優先度 | 重要性 |
| :--- | :--- | :--- |
| **メインクッキー** | 高 | 基礎スコアと能力発動の頻度を決定 |
| **対応するペット** | 中 | コンビボーナススコア / 生存能力を提供 |
| **宝物** | 極めて高 | 合計スコアの倍増 / ユーティリティを提供 |
| **魂の石** | 低 | 特定クッキーの長期的な育成 |

<Callout type="info" title="コンビシナジー">
キャラクターメニューにある「コンビボーナス」の説明を必ず確認してください。クッキーを対応するペットと組み合わせることで、レア度は高くても相性の合わないペットを使用するよりも、はるかに大きなスコア倍率が得られることがよくあります。
</Callout>

</Section>

<Section icon={Swords} title="リーグ戦とハイスコア獲得戦略">

**クッキーラン：クラシックの遊び方**の基本をマスターしたら、本当の挑戦はリーグモードにあります。これは、特定のマップで他のプレイヤーのスコアとリアルタイムで競い合う環境です。リーダーボードの頂点に立つには、素早い指の動きだけでなく、最適化されたルート構築が必要です。

**ハイスコア戦略の4大柱：**
1. **ルートの暗記**: 最も多くのゼリーが配置されている場所や、どのルートを走れば一番スコアが稼げるかを把握します。
2. **障害物の回避**: 「ノーダメージ（ノーヒット）」で完走することが極めて重要です。衝突するたびに体力が減り、ステージクリア時のボーナスを受け取れなくなってしまいます。
3. **アクティブ能力の発動タイミング**: スコア倍率を最大化するために、ゼリーが密集しているエリアでクッキーの能力を発動させます。
4. **宝物の相乗効果**: マップを補完する宝物を使用します。例えば、ゼリーが散らばっているマップでは磁力効果を持つ宝物を使用します。

| 戦略レイヤー | 重点エリア | 期待される成果 |
| :--- | :--- | :--- |
| **操作技術** | ジャンプ/スライドの正確さ | 走行距離の延長と生存率の向上 |
| **戦術** | 能力の管理 | 高速ダッシュ中のピークスコア獲得 |
| **戦略** | コンビの最適化 | 高い基礎スコア倍率の獲得 |
| **リソース** | 宝物のアップグレード | 恒久的なスコア底上げ |

<Callout type="tip" title="プロリーグのヒント">
リーグメニューから上位プレイヤーのプレイ動画（リプレイ）を観戦しましょう。どのタイミングでジャンプしているか、どの宝物を装備しているかに注目してください。これが現在の「メタ」ルートを学ぶ最も手っ取り早い方法です。
</Callout>

</Section>

<Section icon={Shield} title="アカウントの保護とトラブルシューティング">

クッキーラン：クラシックでは、ゲームの進行状況を守ることが非常に重要です。多くの時間を費やし、アプリ内課金を行う可能性もあるため、アカウントへのアクセスを失うことは大きな損失になります。Devsistersは、DevPlayシステムを通じてデータを保護するいくつかの方法を提供しています。

**よくあるトラブルシューティングの手順：**
- **ゲームのクラッシュ**: 端末に少なくとも3GBの空き容量があることを確認してください。バックグラウンドアプリを閉じてRAMを解放します。
- **接続の問題**: リーグ戦でラグが発生する場合は、モバイルデータ通信から安定したWi-Fi接続に切り替えてください。
- **アップデートのエラー**: App StoreまたはGoogle Playで最新バージョンを確認してください。アップデートが表示されない場合は、ストアのキャッシュをクリアしてみてください。

<Steps>
  <Step num="1" title="DevPlayアカウントの作成">
    「設定」 > 「ゲーム情報」に進み、「アカウント連携」を選択します。永続的にアクセスできるメールアドレスを使用してください。
  </Step>
  <Step num="2" title="クラウド保存の有効化">
    連携が完了すると、進行状況は自動的にクラウドに保存されます。これで、どの端末（iOSまたはAndroid）からでもログインしてランを再開できます。
  </Step>
  <Step num="3" title="プレイヤーIDの控え">
    プレイヤーID（設定内に表示）のスクリーンショットを保存しておいてください。アカウント復旧のためにサポートに問い合わせる際に必要となります。
  </Step>
</Steps>

<Callout type="warning" title="ゲストアカウントについて">
「ゲスト」のままプレイするのは危険です。アプリを削除したり、スマートフォンを初期化したりすると、進行状況は永久に失われます。チュートリアル終了後、すぐにアカウントを連携してください。
</Callout>

</Section>
