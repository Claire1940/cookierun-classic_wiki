<Section icon={Target} title="基本的なランニング操作のマスター">

ゲームプレイは、レスポンスの良いクラシックな2ボタン操作に基づいています [homepage-modules-1]。画面の左側にはジャンプボタン、右側にはスライドボタンがあります [homepage-modules-1]。シンプルな操作ですが、グローバルランキングでハイスコアを出すには、正確なタイミング、一瞬の判断力、そしてキャラクターの当たり判定の理解が求められます [homepage-modules-2]。

| アクション | 主な機能 | スコア最適化戦略 |
| :--- | :--- | :--- |
| ジャンプ | 地面の障害物やトゲ、穴を飛び越える | 2回タップして2段ジャンプを行い、高い場所にあるゼリーを回収する |
| スライド | 低い天井や飛んでくる障害物の下をくぐる | スライドボタンを押し続けて、長いトンネルを通り抜ける |
| ゼリー回収 | ポイントを蓄積してスコアを伸ばす | ゼリーの配置パターンを覚え、取りこぼしなく全回収を目指す |

走っている間、クッキーの体力バーは絶えず減少していきます [base-info]。障害物に衝突すると体力が大幅に減り、一時的に移動速度が低下します [homepage-modules-1, homepage-modules-2]。より長く走りスコアを伸ばすには、ジャンプとスライドを正確にこなしながら、ステージ内に配置された回復ポーションを回収する必要があります [base-info, homepage-modules-2]。

<Callout type="warning" title="衝突によるペナルティ">
障害物に当たると体力が減るだけでなく、クッキーのアクティブスキルが一時的に無効化され、ペットの能力発動タイマーも乱されます [homepage-modules-2]。ハイスコア倍率を維持するため、リスクのあるゼリー回収よりも、生存と障害物回避を優先しましょう [homepage-modules-2]。
</Callout>

<Section icon={Shield} title="リーグ対戦とアカウントのセキュリティ">

ゲームの核心となるエンドコンテンツは、リアルタイムのリーグモードです。ここでは、プレイヤー全員が同じトラックで競い合い、最高スコアを目指します [homepage-modules-1, homepage-modules-2]。ランクを上げることで、レアなクッキーのソウルストーン、クリスタル、ゴールドが入った宝箱を獲得できます [homepage-modules-1, homepage-modules-2]。

競争の記録を守るために、すぐにアカウントを保護する必要があります [homepage-modules-1]。ゲストアカウントでのプレイは非常に不安定です。アプリの削除、スマートフォンの初期化、あるいはシステムクラッシュが発生した場合、進行状況は永久に失われます [homepage-modules-2]。
