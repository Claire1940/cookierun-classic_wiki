<Section icon={Target} title="クッキーラン：クラシックのクーポンコードと入力ガイド">

**クッキーラン：クラシック**のコード入力は、DevPlayポータルを通じて外部で行われます。これにより、使用しているデバイスに関係なく報酬がアカウントに同期されます。公式X（旧Twitter）やInstagramなどのSNSで公開されるクーポンコードを利用するには、報酬をゲーム内のメールボックスに確実に届けるために、以下の手順に従う必要があります。

**動画のハイライト:**
- DevPlayクーポン入力サイトのステップバイステップ解説。
- ゲーム設定内でのプレイヤーIDの確認方法。
- リリース記念ギフト報酬の確認。

  <Step num="1" title="プレイヤーIDを確認する">
    クッキーラン：クラシックを開き、**設定 > ゲーム情報**に移動します。そこに表示されているDevPlayメールアドレスまたは固有のプレイヤーID文字列をコピーします。
  </Step>
  <Step num="2" title="クーポンサイトにアクセスする">
    公式の [DevPlayクーポンページ](https://coupon.devplay.com/coupon/crg/en) にアクセスします。これはクッキーラン：クラシックの報酬を受け取ることができる唯一の公式サイトです。
  </Step>
  <Step num="3" title="詳細を入力する">

  </Step>
  <Step num="4" title="アプリを再起動する">
    「クーポン利用」をクリックした後、ゲームを完全に終了して再起動します。数分以内に報酬が**ゲーム内のメールボックス**に届きます。

</Callout>

<Section icon={Trophy} title="最強クッキー・ペット＆コンビティアリスト">

**クッキーラン：クラシック**における効率は「コンビ」によって決まります。これはクッキー、ペット、そして3つの宝物の組み合わせです。スコアの可能性を最大化したり、コイン稼ぎを効率化したりするには、各要素が相乗効果を発揮する必要があります。2026年のメタでは、高い機動力を持つクッキーと、特殊なゼリーを生成するペットが好まれています。

| ティア | クッキー | ペット | 最適な用途 |
| :--- | :--- | :--- | :--- |
| **S** | **蓮根の幽霊味クッキー** | **ハスのお世話係** | 競技リーグ / ハイスコア |
| **S** | **松の僧侶味クッキー** | **イベント限定ペット** | 現在の「お出かけパーティー」イベント |

| **A** | **K-Culture限定クッキー** | **対応ペット** | テーマ別スコアリング |
| **B** | **初期クッキー** | **基本ペット** | 初期の資源集め |

リリース直後の期間では、**蓮根の幽霊味クッキー**と**ハスのお世話係**が際立ったパフォーマンスを見せています。マップの広い範囲をクリアしながら高価値のゼリーを生成する能力により、リーグのリーダーボードを目指す上で最適な選択肢となります。

| 宝物名 | 効果 | アップグレード優先度 |
| :--- | :--- | :--- |

| **黄金のコイン磁石** | コインの引き寄せ範囲を拡大 | 中（稼ぎ） |
| **スコアブーストリボン** | スコアに一定のパーセンテージ加算 | 極めて高（リーグ） |

<Section icon={Shield} title="アカウント引き継ぎとトラブルシューティング">

特に2026年に予定されている頻繁なアップデートを考慮すると、**クッキーラン：クラシック**の進行状況を保護することは不可欠です。このゲームはライブサービスであるため、セーブデータはDevsistersのサーバーに保存されますが、複数のデバイスでアクセスするには永続的なIDに連携されている必要があります。

| 問題のカテゴリ | 要件 | 解決策 |
| :--- | :--- | :--- |
| **ストレージ** | 3GB - 4GB の空き容量 | 未使用アプリの削除またはキャッシュ消去 |
| **OSバージョン** | iOS 15+ / Android 8.0+ | 設定からシステムソフトウェアを更新 |
| **接続** | 安定した Wi-Fi/LTE | リーグ挑戦中は公共VPNの使用を避ける |
| **アカウント紛失** | DevPlay/Apple/Google連携済み | プレイヤーIDを添えてサポートへ連絡 |

新しいデバイスに移動する場合は、ゲームをインストールし、古いデバイスで使用していたのと同じアカウントプロバイダー（Apple、Google、またはDevPlay）でログインするだけです。クッキー、ペット、購入したクリスタルを含むすべての進行状況が自動的に同期されます。

<Steps>
  <Step num="1" title="連携を確認する">

  </Step>
  <Step num="2" title="新しいデバイスを準備する">
    公式のApp StoreまたはGoogle Playストアからクッキーラン：クラシックをインストールします。
  </Step>
  <Step num="3" title="データを同期する">
    タイトル画面で「ログイン」を選択し、同じ資格情報を使用します。
  </Step>
</Steps>

<Callout type="warning" title="最低動作環境">
古いデバイスでもインストールは可能ですが、**リーグでのパフォーマンス**においてフレーム落ちが発生する可能性があります。競技プレイのためには、少なくとも4GBのRAMと安定した5GHz Wi-Fi接続を確保することをお勧めします。
</Callout>
