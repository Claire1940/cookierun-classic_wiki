<Section icon={Target} title="食材集めルート＆獲得場所">

**クッキーランクラシック レシピブック**に登場する上位の料理を作るには、ギルドで効率的に食材を集める必要があります。基本となる食材は6種類あり、それぞれ特定の探検ステージからドロップします。

### 食材ドロップ場所

| 食材 | 主な探検ステージ | レア度 |
| :--- | :--- | :--- |
| **氷シダ** | 薄暗い森、凍てつく山荘 | 一般 |
| **ミミズゼリー** | 竜の渓谷への入口、謎の海岸 | 一般 |
| **水トカゲのしっぽ** | 凍てつく山荘、謎の海岸 | 中間 |
| **スパイシージャム** | 竜の渓谷への入口 | 中間 |
| **気絶した高麗人参** | 黒い骸骨の荒野 | レア |
| **未熟なイチゴ** | 黒い骸骨の荒野、薄暗い森 | レア |

毎日の食材獲得量を最大化するために、ギルドメンバーと協力して以下の最適化された収集ワークフローを実行しましょう。

<Steps>
  <Step num="1" title="ギルド派遣の調整">
    ギルドリーダーは、メンバーごとに特定の探検ゾーンを割り当てましょう。これにより、氷シダのような一般の食材が過剰に集まる一方で、レアなドロップ食材が不足するのを防ぐことができます。
  </Step>
  <Step num="2" title="レアゾーンを最優先にする">
    レベルの高いクッキーを優先的に「黒い骸骨の荒野」へ派遣しましょう。気絶した高麗人参と未熟なイチゴは、ほぼすべての最上位レシピのボトルネック（不足しがちな食材）となります。
  </Step>
  <Step num="3" title="毎日報酬を受け取り、寄付する">
    探検が完了したら、すぐに報酬を受け取りましょう。余った食材は常にギルドキッチンの倉庫に寄付し、リーダーやサブリーダーが次の主要な料理をすぐに調理できるように準備しておきます。
  </Step>
</Steps>

<Callout type="warning" title="探検回数の制限">
各プレイヤーの1日の派遣回数には上限があります。ギルドにすでに十分な量がある食材のゾーンに派遣を浪費しないでください。派遣する前に、必ずギルドチャットや掲示板を確認しましょう。
</Callout>

</Section>

<Section icon={Sparkles} title="料理戦略とランの最適化">

レシピを調理することは戦い半分に過ぎません。最適な結果を得るためには、アクティブなギルドバフとランナーの編成を一致させる必要があります。クッキーとペットの組み合わせによって、恩恵を受けられるレシピバフが異なります。

例えば、アクティブなギルドレシピが**渋いサラダ**（障害物破壊スコア +15,000点）の場合、障害物破壊に特化したチームを編成すべきです。障害物を吹き飛ばすクッキーや宝物を使用することで、何百万もの追加スコアを獲得できます。逆に、**冷たいスープ**（クマゼリースコア +1,600点）が有効なときに、純粋なコイン稼ぎチームを走らせるのは非常に非効率的です。
