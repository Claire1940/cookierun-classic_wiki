<Section icon={Target} title="ステップバイステップのリセマラ手順">

完璧なリセマラを実行するには、不本意なアカウントを永続的なメールアドレスに連携させないよう、特定の順序に従う必要があります。以下の手順に従って、希望のユニットが出るまで効率的にリセットを繰り返してください。

<Steps>
  <Step num="1" title="ゲストとして開始">
    公式App StoreまたはGoogle Playからクッキーランクラシックをダウンロードします。DevPlayのログインプロンプトが表示されたら、**ゲストログイン**を選択します。これにより、メールアドレスを連携させずにプレイを開始できます。
  </Step>
  <Step num="2" title="基本チュートリアルを完了">
    画面の指示に従って、ジャンプとスライドの操作を学びます。チュートリアルは比較的短く、ゼリーを食べたり魔女の障害物を避けたりすることに焦点を当てています。このセクションを完了してメインロビーをアンロックします。
  </Step>
  <Step num="3" title="リリース報酬を受け取る">
    ゲーム内のメールボックスまたはグランドオープンイベントタブへ移動します。**300クリスタル、100,000コイン**、および**3枚の最高級宝物チケット**を受け取ります。これらがリセマラの主要なツールとなります。
  </Step>
  <Step num="4" title="プレミアムガチャを引く">
    ガチャショップへ向かいます。クリスタルをピックアップクッキー/ペットバナー（現在は**蓮根幽霊味クッキー**）に使用します。最高級宝物チケットを使用して、高レアリティの宝物を狙います。
  </Step>
  <Step num="5" title="評価またはリセット">
    S級のクッキーや宝物が出なかった場合は、**設定 > ゲーム情報**に移動します。**アカウント削除**または**ログアウト**のオプションを探します。Androidの場合は、デバイスの設定でアプリのキャッシュ/データを消去してゲストアカウントをリセットすることも可能です。
  </Step>
</Steps>

<Callout type="warning" title="データ消失の警告">
ガチャの結果に100%満足するまで、アカウントをGoogleやApple IDに連携させないでください。一度連携するとリセットが難しくなり、別のメールアドレスが必要になる場合があります。
</Callout>

| リセマラ段階 | 推定時間 | 重点項目 |
| :--- | :--- | :--- |
| **チュートリアル** | 3-5分 | 操作の習得と基本メカニクス。 |
| **報酬受け取り** | 1分 | 全てのイベントおよびリリース記念メールの回収。 |
| **ガチャ** | 2分 | クッキー、ペット、宝物の獲得。 |
| **リセット** | 1分 | データの消去またはゲストプロフィールの削除。 |

</Section>

<Section icon={Sparkles} title="宝物ガチャの最適化">

宝物はクッキーランクラシックにおける影の立役者です。マップに応じて入れ替えるクッキーとは異なり、高品質な宝物は複数のチームで使い回すことができ、パッシブなスコアブースト、体力回復、磁力効果などを提供します。

| 宝物の種類 | 用途 | リセマラ推奨度 |
| :--- | :--- | :--- |
| **スコアブースター** | リーグ戦 | 競技プレイヤーには**必須**。 |
| **磁力装備** | リソース収集 | ゼリーの取りこぼしを防ぐため、初心者には最適。 |
| **体力/シールド** | 長距離走行 | 障害物の多い難しいマップをクリアするのに有用。 |

配布される**3枚の最高級宝物チケット**を賢く使うことが不可欠です。メインのクッキーを補完するエピックまたはレジェンダリー級の宝物を引けた場合は、そこでリセマラを終了することを検討すべきです。宝物はアップグレード可能であり、初期に運良く引けた宝物はコインを投資するにつれてさらに強力になっていきます。

<Section icon={Shield} title="アカウントの確定と保護">

**クッキーランクラシック リセマラガイド**を参考に満足のいく結果が得られたら、データを保護する必要があります。ゲストとしてプレイし続けるのは危険です。スマホを紛失したり、誤ってアプリを削除したりすると、進行状況は永遠に失われます。

アカウントを連携させることで、複数のデバイスでプレイすることも可能になります。Androidのエミュレータでリセマラを行い、iPhoneでプレイしたい場合、DevPlay IDがクッキーたちを引き継ぐ架け橋となります。

| 連携方法 | メリット | デメリット |
| :--- | :--- | :--- |
| **DevPlayメール** | 共通アクセス、復旧が容易 | 別のパスワードを覚える必要がある。 |
| **Google/Apple ID** | ワンタップログイン、非常に安全 | OSプラットフォーム間の切り替えが難しい場合がある。 |
| **ゲストアカウント** | リセマラが最速 | データ全損失の**高いリスク**。 |

<Callout type="warning" title="特典の注意点">
連携後は、公式のDevPlayクーポンページをチェックして、有効なコードがないか確認してください。リリース直後に確認されていない場合でも、公式X（旧Twitter）アカウントで新しいコードが投稿されることがよくあります。
</Callout>
