<Section icon={Sparkles} title="アカウント保護＆不具合発生時の対処法">

ゲームのハイスコアやキャラクターの育成状況を失わないために、アカウントの進行状況を保護することは非常に重要です [1]。**「クッキーランクラシック グローバル」**では、DevPlay、Google、Apple IDを介した強力なクラウド保存機能が提供されています [1]。

動作の重さ、強制終了、インストールエラーなどの問題が発生した場合は、以下の手順を試して動作を安定させてください [1]。

| 発生する問題 | 主な原因 | 推奨される対処法 | リスクレベル |
| :--- | :--- | :--- | :--- |
| ゲームの強制終了（iOS） | メモリ不足 / OSのバージョンが古い | iOSのアップデート、2〜3 GBの空き容量確保、端末の再起動 | 中 |
| ゲームの強制終了（Android） | メモリ不足 / OSのバージョンが古い | Androidのアップデート、4 GBの空き容量確保、端末の再起動 | 中 |
| 接続エラー | ネットワークが不安定 | 安定したWi-Fiまたはモバイルデータ通信に切り替える | 低 |
| インストール失敗 | ストレージ容量不足 | 空き容量を増やす、公式ストアのみを使用する | 高 |

<Callout type="warning" title="ゲストアカウントに関する警告">
ゲストアカウントでのプレイは、ゲームデータの紛失リスクが非常に高くなります [1]。DevPlay、Google、またはApple IDに連携せずにアプリを削除したり、端末を初期化したりした場合、アカウントを復旧することはできません [1]。ゲームを開始したら、すぐに設定メニューからアカウントを連携してください [1]。
</Callout>
