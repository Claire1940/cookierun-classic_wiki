<Section icon={Target} title="コードの引き換えとギフトの受け取り方法">

**クッキーランクラシック事前登録**報酬やその後のイベントギフトの受け取りには、ゲーム内のイベントタブと外部のDevPlayクーポンポータルの2つの経路があります。ほとんどのリリースギフトはチュートリアル完了後にゲーム内のメールボックスに直接届きますが、特定のプロモーションコードは手動で入力する必要があります。

DevPlayシステムは、すべてのDevsistersタイトルの中心的なハブです。報酬を確実に受け取るためには、ゲームアカウントを有効なメールアドレス、Google、またはApple IDに連携させる必要があります。ゲストアカウントは復旧が非常に困難であり、クーポン引き換えプロセスで問題が発生することがよくあります。

<Steps>
  <Step num="1" title="プレイヤーIDを確認する">
    クッキーランクラシックを開き、「設定」メニューに移動します。「ゲーム情報」タブの中に、DevPlayアカウント（メールアドレス）またはプレイヤーIDが表示されています。これを表示通りにコピーしてください。
  </Step>
  <Step num="2" title="クーポンサイトにアクセスする">
    公式サイトのDevPlay引き換えページ（`https://coupon.devplay.com/coupon/crg/ja`）にアクセスします。これはプロモーションコードを入力するための唯一の公式サイトです。
  </Step>
  <Step num="3" title="詳細とコードを入力する">
    最初のフィールドにプレイヤーIDを貼り付け、2番目のフィールドに特定のクーポンコードを入力します。システムは形式に敏感なため、前後に余計なスペースが入らないように注意してください。
  </Step>
  <Step num="4" title="アプリを再起動する">
    フォームを送信してゲームに戻ります。報酬をサーバーと同期させ、メールボックスに反映させるには、アプリを完全に終了してから再起動する必要があります。
  </Step>
</Steps>

<Callout type="warning" title="引き換え時の注意">
各クーポンコードは1アカウントにつき1回のみ使用可能です。エラーメッセージが表示された場合は、コードの期限が切れていないか、プレイヤーIDが正しく入力されているかを確認してください。公式コードは通常、@CRClassicJP（またはグローバル版公式）のXアカウントを通じて配布されます。
</Callout>

| 引き換え時の問題 | 主な原因 | 推奨される解決策 |
| :--- | :--- | :--- |
| **無効なID** | 入力ミスまたはゲストアカウント | 最初にアカウントをDevPlayに連携する |
| **コード期限切れ** | 期間限定イベントの終了 | 公式Xで有効なコードを確認する |
| **報酬が表示されない** | 同期の遅延 | アプリを再起動してメールボックスを確認 |
| **リージョンロック** | グローバル版では稀 | 適切な言語ポータルを使用しているか確認 |

<Section icon={Shield} title="アカウントの安全性とテクニカルサポート">

モバイルファーストの体験として、クッキーランクラシックをスムーズに動作させるには、安定したインターネット接続と特定のハードウェア基準が必要です。**クッキーランクラシック事前登録**に参加した多くのプレイヤーでも、古いOSや脱獄（Jailbreak）されたデバイスでプレイしようとすると問題が発生する可能性があります。

Devsistersは、2026年のリリースに向けて堅牢なトラブルシューティングの枠組みを提供しています。最も一般的な問題は、インストールの失敗とアカウントの同期に関連するものです。デバイスに少なくとも3〜4GBの空きストレージがあることを確認することが、ほとんどのクラッシュ関連の問題を解決するための第一歩です。

| プラットフォーム | 最低OS | 推奨ストレージ |
| :--- | :--- | :--- |
| **iOS / iPadOS** | バージョン 15.0+ | 3 GB の空き容量 |
| **Android** | バージョン 8.0+ | 4 GB の空き容量 |
| **接続環境** | 安定した Wi-Fi/LTE | 低レイテンシが必要 |

<Callout type="warning" title="データ紛失の防止">
DevPlay、Google、またはAppleアカウントを連携せずにアプリをアンインストールすると、進行状況が永久に失われます。サポートチームは、ローカルストレージから削除されたゲストアカウントを復旧することはできません。
</Callout>

持続的なバグや購入の失敗に遭遇した場合は、「ゲーム情報」タブにあるゲーム内の「サポート」ボタンを使用してください。これによりデバイスのログが自動的にチケットに添付され、技術チームが問題をより迅速に診断できるようになります。クーポン関連の失敗については、必ずエラーメッセージのスクリーンショットとプレイヤーIDを添えてください。
