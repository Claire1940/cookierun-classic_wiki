<Section icon={Target} title="基本ゲームプレイと初心者向け戦略">

**クッキーランクラシック モバイル 1.0 グローバルリリース**後のゲームプレイは、オリジナル版のシステムを忠実に再現しています。主な目的は、障害物を避けながらゼリーを集め、できるだけ遠くまで走ることです。「クラシック」版では、複雑なスキルの発動手順よりも正確な操作が重視されるため、初心者でも簡単に始められますが、高レベルのリーグで上位を目指すには熟練が必要です。

| 操作・システム | 説明 | 攻略のコツ |
| :--- | :--- | :--- |
| **ジャンプ** | 床 of 障害物や穴を飛び越えます | 2回タップ（2段ジャンプ）すると、積み上げられた箱を高く飛び越えられます。 |
| **スライド** | 頭上にある障害物の下をくぐり抜けます | 長押しして低い姿勢を維持し、素早く離して走りに戻ります。 |
| **ゼリー** | 主なスコア獲得源です | ゼリーの配置に沿って進むのが、ステージを安全に進むルートになります。 |
| **体力（エネルギー）** | 時間の経過や障害物への衝突で減少します | より長く走るために、珍しいゼリーよりも**回復ポーション**の獲得を優先しましょう。 |
| **コンビボーナス** | クッキーとペットのシナジー効果です | 特定のペットと相性の良いクッキーを組み合わせることで、追加のボーナススコアを獲得できます。 |

<Callout type="info" title="リーグ競争">
リアルタイムのリーグ戦は、本作の競争要素の核となります。世界中の同じ実力を持つプレイヤーとマッチングし、週替わりでローテーションするステージでハイスコアを競い合います。
</Callout>
