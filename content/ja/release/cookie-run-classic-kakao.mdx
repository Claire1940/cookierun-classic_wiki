<Section icon={History} title="クッキーランクラシック Kakaoの軌跡">

**クッキーランクラシック Kakao**（韓国語：쿠키런: 쿠키들의 오븐탈출 대작전！）は、クッキーランの世界観の原点です。2013年4月2日にリリースされたこのバージョンは、『オーブンブレイク』や『キングダム』といったすべての後続タイトルの基礎となっています。2018年にサービス終了したグローバルLINE版とは異なり、Kakao版は一貫したアップデートと韓国国内の熱心なプレイヤー層に支えられ、今なお活発なエコシステムを維持しています。

ゲームはKakaoTalkのソーシャルプラットフォーム上で動作しており、そのソーシャル機能の多くがゲームプレイに直結しています。ジャンプとスライディングというコアな操作性は一貫しているものの、Kakao版は新しいシステムやシーズンコンテンツをシリーズの他タイトルに導入する前のテストグラウンドとしての役割を果たすことがよくあります。2026年現在、同地域で最も長く運営されているアクティブなモバイルランナーゲームとなっています。

| 特徴 | Kakao版の詳細 |
| --- | --- |
| **リリース日** | 2013年4月2日 |
| **プラットフォーム** | KakaoTalk（韓国） |
| **ステータス** | サービス中（2026年継続中） |
| **対応言語** | 主に韓国語 |
| **ソーシャル連携** | KakaoTalkの友だちリスト |

<Callout type="warning" title="地域制限について">
韓国以外の地域に住むプレイヤーは、韓国のApple IDやGoogle Playアカウントがないとゲームにアクセスするのが難しい場合があります。また、データの保存やソーシャル機能の利用には、認証済みのKakaoTalkアカウントが必須となります。
</Callout>
